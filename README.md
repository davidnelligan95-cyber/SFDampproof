
# SF Damp Surveying & Remedial Works — Website

This is a small, static website scaffold for SF Damp Surveying & Remedial Works. It includes `index.html`, `services.html`, and `contact.html`.

Quick setup:

1. Replace the placeholder logo at `assets/logo.svg` or `assets/stevo-logo-2.png` with your real logo.
2. Update contact phone/email in the HTML files.
3. Configure the contact form: open `contact.html` and replace `https://formspree.io/f/your-form-id` with your Formspree form endpoint or another endpoint.
4. Preview locally by opening `index.html` in a browser.
5. Deploy: push to GitHub and use GitHub Pages, or drag the folder to Netlify/Vercel for one-click deploy.
